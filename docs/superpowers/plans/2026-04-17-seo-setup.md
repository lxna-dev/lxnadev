# SEO Setup Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add full SEO coverage to lxna.dev — metadata, dynamic OG image, robots.txt, sitemap.xml, and JSON-LD structured data — using Next.js 15 native APIs with zero new dependencies.

**Architecture:** All SEO is handled through Next.js App Router conventions: the `metadata` export in `layout.tsx` populates `<head>`, route handler files (`robots.ts`, `sitemap.ts`, `opengraph-image.tsx`) are auto-discovered by the framework and served at their conventional URLs. JSON-LD is injected as a raw `<script>` tag in the root layout body using a hardcoded constant (no XSS risk).

**Tech Stack:** Next.js 15 App Router, `next/og` (ImageResponse — built-in), TypeScript

> **Note:** No test suite is configured for this project. Verification steps use the dev server and direct URL checks instead of automated tests.

### Task 2: Add JSON-LD structured data to `layout.tsx`

**Files:**
- Modify: `src/app/layout.tsx`

- [ ] **Step 1: Add JSON-LD script tag to the root layout body**

In `src/app/layout.tsx`, inside `RootLayout`, define the JSON-LD constant and inject it as a `<script>` tag. The content is a hardcoded constant — no XSS risk.

```tsx
export default function RootLayout({
  children,
}: Readonly<{
  children: React.ReactNode;
}>) {
  const jsonLd = {
    "@context": "https://schema.org",
    "@type": "Person",
    name: "Beljohn Luna",
    url: "https://www.lxna.dev",
    jobTitle: "Full Stack Developer",
    email: "beljohnluna@gmail.com",
    sameAs: [
      "https://github.com/lxna-dev",
      "https://www.linkedin.com/in/beljohn-luna/",
    ],
  };

  return (
    <html lang="en">
      <body
        className={`${geistSans.variable} ${geistMono.variable} ${unifrakturMaguntia.variable} ${voltaire.variable} ${syneMono.variable} mx-auto antialiased`}
      >
        <script
          type="application/ld+json"
          // Content is a hardcoded constant — safe to use dangerouslySetInnerHTML here
          dangerouslySetInnerHTML={{ __html: JSON.stringify(jsonLd) }}
        />
        <SidebarLayout>{children}</SidebarLayout>
      </body>
    </html>
  );
}
```

- [ ] **Step 2: Verify TypeScript compiles**

Run: `bun build`
Expected: build succeeds with no errors

- [ ] **Step 3: Verify JSON-LD appears in page source**

Run: `bun dev`, open `http://localhost:3000`, view page source (Ctrl+U).
Expected: `<script type="application/ld+json">` tag present in `<body>` containing the Person schema JSON
